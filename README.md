## Descrição

Este projeto é a interface de usuário (UI) de uma loja virtual, desenvolvida em TypeScript. O objetivo é oferecer uma experiência moderna e eficiente para lojas online, facilitando a navegação e a compra de produtos.

## Principais Funcionalidades

- Visualização de produtos em catálogo
- Carrinho de compras interativo
- Página de detalhes do produto
- Processo de checkout
- Integração com backend para produtos, usuários e pedidos (em progresso)

## Tecnologias Utilizadas

- **TypeScript**
- Framework moderno de frontend (não especificado)
- Deploy em [Vercel](https://virtual-store-ui.vercel.app)
